- The catch-all route matches multiple segments or a single segment. It’s useful when you want to match an arbitrary depth of nested paths.

- **Example**: `/catch-all-segments/[...slug]` matches `/catch-all-segments/intro`, `/catch-all-segments/guides/installation`, or even `/catch-all-segments/guides/installation/linux`.

## Folder Structure:

```javascript
app
└── catch-all-segments
    └── [...slug]
        └── page.tsx
```

- The `slug` variable can be used to fetch the data from the server to render content for the page.

```tsx
const res = await fetch(
  `https://example.com/api/catch-all-segments/${slug.join("/")}`
);
const data = await res.json();
```
